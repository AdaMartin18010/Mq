## 1.20.1 场景速查

### 按场景快速选型

| 场景 | 推荐系统 | 备选系统 |
|------|---------|---------|
| **日志聚合** | Kafka | Pulsar |
| **IoT设备** | MQTT | NATS |
| **微服务通信** | NATS | RabbitMQ, Pulsar |
| **任务队列** | RabbitMQ | NATS, Redis Stream |
| **事件流处理** | Kafka | Pulsar |
| **实时通信** | NATS | Redis Stream |
| **大数据分析** | Kafka | Pulsar |
| **移动推送** | MQTT | NATS |

## 1.20.2 性能速查

### 性能需求快速匹配

| 需求 | 推荐系统 | 性能指标 |
|------|---------|---------|
| **最高吞吐量** | NATS Core | 200万+ msg/s |
| **高吞吐量+持久化** | Kafka | 120万+ msg/s |
| **最低延迟** | NATS Core | <100μs |
| **低延迟+持久化** | Redis Stream | <5ms |
| **高并发连接** | NATS/MQTT | 10万+ |

## 1.20.4 快速决策

### 决策流程图

```
需要持久化？
├─ 是 → 需要高吞吐量？
│   ├─ 是 → Kafka或Pulsar
│   └─ 否 → RabbitMQ或Redis Stream
└─ 否 → 需要低延迟？
    ├─ 是 → NATS Core
    └─ 否 → MQTT或NATS
```

### 一句话选型

- **日志/事件流** → Kafka
- **IoT设备** → MQTT
- **微服务实时通信** → NATS
- **任务队列** → RabbitMQ
- **轻量级流处理** → Redis Stream
- **云原生多租户** → Pulsar
