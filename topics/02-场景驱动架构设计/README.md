## 主题概述

本主题从场景驱动的角度分析消息队列架构设计，包括：

1. **场景化功能架构矩阵**：不同场景下的技术选型和架构论证
2. **多场景架构决策图网**：动态决策网络、场景权衡、演进路径
3. **程序设计模式场景化应用**：电商、IoT、金融等场景的设计模式实践

## 核心场景

- **日志聚合**：Kafka高吞吐+持久化
- **IoT设备**：MQTT轻量+QoS分级
- **微服务通信**：NATS低延迟+自愈
- **事件溯源**：Kafka不可变日志
- **边缘计算**：NATS轻量部署

## 学习路径

1. 理解场景特征和关键需求
2. 学习场景化架构设计方法
3. 掌握多场景决策框架
4. 实践程序设计模式应用
